# PhotoSurfPaysBasque

Site officiel de **PhotoSurfPaysBasque** — photographe & vidéaste spécialisé océan à Anglet, Biarritz et dans le sud des Landes.

🌐 [photosurfpaysbasque.fr](https://photosurfpaysbasque.fr)

## Stack

- React 18 + Vite
- TypeScript
- Tailwind CSS
- shadcn/ui

## Développement local

```sh
npm install
npm run dev
```

## Build production

```sh
npm run build
```
